# Ignition-Athera

Project contains the following files.

# Micro Python code for Pico-W

**File** : gps.py 
**Description** : This file has the logic for receiving the GPS data from the uBlox module.
**Usage** : Open the file in Thonny editor. Drag the file to Pico-W

**File** : mpu.py 
**Description** : This file has the logic for receiving the accelerometer, gyro and magnetometer data from the IMU module.
**Usage** : Open the file in Thonny editor. Drag the file to Pico-W


# Web Client using NextJS

**Folder** : web
**Description**: 
**Usage** : Launch VS Code and open the web folder. Execute 'npm install' command followed by 'npm run dev' to start the program.


# Web Start using NodeJS

**Folder** : server
**Description**: 
**Usage** : Launch VS Code and open the web folder. Execute 'npm install --legacy-peer-deps' command followed by 'npm start' to start the program.
